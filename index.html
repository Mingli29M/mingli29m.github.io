PixelStudio

PixelStudio is a full-featured, open-source pixel art application built with Godot 4.x. Designed as an alternative to Aseprite, it includes drawing tools, layer management, animation timeline, color palettes, and more!

Key Features:
  • Complete drawing tool suite
  • Layer system with blend modes
  • Frame-by-frame animation
  • Custom color palettes
  • Export to PNG, GIF, sprite sheets
